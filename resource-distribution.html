Minecraft 1.16.4 resource distrubution by altitude

Plugin and datapack

The plugin used to retrieve this data can be found here. A precompiled JAR can be found here.

BlueCommander's World Loader V2 datapack was used in conjunction with the plugin to scan 200,298 chunks with a total of 13,126,729,728 blocks. A video covering the datapack can be found here.

The raw data collected by the plugin is available as a CSV file and as a table. Note that the data is not sorted in any particular order.

This project was inspired by a graph created by Orthotope in 2013 and a tool created by TR of Digital-Trauma.de in 2012.
