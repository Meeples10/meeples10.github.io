Minecraft 1.16.4 resource distrubution by altitude

Plugin and datapack

The plugin used to retrieve this data can be found here. A precompiled JAR can be found here.

BlueCommander's World Loader V2 datapack was used in conjunction with the plugin to scan 1,236,393 chunks with a total of 81,028,251,648 blocks (22,201,992,577 excluding air) from a world generated with the seed -3569932767750160641. A video covering the datapack can be found here.

The raw data collected by the plugin is available as a CSV file (Overworld, Nether, End) and as a table (Overworld, Nether, End). Note that the data is not sorted in any particular order.

This project was inspired by a graph created by Orthotope in 2013 and a tool created by TR of Digital-Trauma.de in 2012.
