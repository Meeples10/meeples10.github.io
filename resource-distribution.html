Minecraft 1.16 resource distrubution by altitude

Program and datapack

The program used to retrieve this data can be found here. A precompiled JAR can be found here.

BlueCommander's World Loader V2 datapack was used in conjunction with the plugin to scan 2,286,875 chunks with a total of 147,740,213,248 blocks (27,131,513,691 excluding air) from a world generated with the seed 948029238 (for the overworld) and a world with the seed -3569932767750160641 (for the Nether and End). A video covering the datapack can be found here.

The raw data collected by the program is available as a CSV file (Overworld, Nether, End) and as a table (Overworld, Nether, End).

This project was inspired by a graph created by Orthotope in 2013 and a tool created by TR of Digital-Trauma.de in 2012.

If you have any questions, you can contact me directly or create a GitHub issue.
