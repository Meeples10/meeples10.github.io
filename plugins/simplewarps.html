SimpleWarps

A plugin for creating warps.

Downloads

  • GitHub
  • SpigotMC

Commands

Command Description Permission
/warp Warps the player to a location. simplewarps.warp
/setwarp Sets a warp. simplewarps.setwarp
/delwarp Deletes a warp. simplewarps.delwarp
/warps Lists all available warps. simplewarps.warps

Configuration

The default configuration file can be found here.

Key Description
play-sound If true, a sound will be played when the player is warped.
teleport-sound If play-sound is true, this sound will be played when a player uses /warp. See the Minecraft Wiki for a list of available sounds.
reset-velocity If true, the player's velocity will be reset when teleporting. This prevents fall damage if the player uses /warp while falling.
messages Messages displayed by the plugin.
