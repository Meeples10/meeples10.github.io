LastSeen

A plugin that allows you to view when a player was last online.

Downloads

  • GitHub
  • SpigotMC

Commands

Command Description Permission
/seen <name or UUID> Shows when a player was last online. lastseen.use

Use the permission lastseen.location to also show the player's last location when using /seen.

Configuration

The default configuration file can be found here.
