Minecraft resource distrubution by altitude

The program used to retrieve this data can be found here. Precompiled JARs can be downloaded from the Releases page.

For versions 1.16 to 1.18, this datapack was used to generate large numbers of chunks for analysis.

Unless otherwise specified, chunks for every version from Beta 1.3 onward were generated using the seed gargamel.

All values within the raw data have been rounded to have no more than 10 digits after the decimal.

Note that due to the way Minecraft stores chunks, counts for air may be inaccurate at high altitudes.

This project was inspired by a graph created by Orthotope in 2013 and a tool created by “TR” of Digital-Trauma.de in 2012.

If you have any questions, you can contact me directly or create a GitHub issue.
